# SQL-LIBRERIA

## Descripción
Este repositorio contiene scripts SQL creados para la gestión de un centro de estética. Incluye la creación de bases de datos, tablas y consultas para manejar clientes, empleados, tratamientos, citas y pagos.

## Propósito
Este repositorio ha sido creado con el propósito de **practicar y mejorar habilidades en SQL**. Es un proyecto personal para experimentar y aprender.

## Estructura del Proyecto
- `basededatos.sql`: Contiene la creación de la base de datos y las tablas necesarias.
- `consultas.sql`: Contiene consultas SQL para operaciones básicas, agregación, agrupación y joins.

## Uso
1. **Crear la base de datos y las tablas**:
   - Ejecutar el script `basededatos.sql` en tu servidor SQL para crear la base de datos y las tablas necesarias.

2. **Realizar consultas**:
   - Ejecutar el script `consultas.sql` para realizar diversas operaciones y consultas sobre la base de datos.
